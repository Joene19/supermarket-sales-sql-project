# 🛒 Supermarket Sales SQL Project

## 📌 Overview

This project analyzes supermarket sales data using MySQL. The dataset includes order details such as customers, prices, quantities, taxes, and total sales.

The goal is to practice SQL data analysis skills using real-world structured data.

---

## 🛠 Tools Used

* MySQL
* Excel (CSV format)
* SQL (Data analysis)

---

## 📊 Dataset Information

The dataset contains the following fields:

* Order No
* Order Date
* Customer Name
* Ship Date
* Retail Price (USD)
* Order Quantity
* Tax (USD)
* Total (USD)

---

## 🔍 Analysis Performed

This project includes:

* Total revenue calculation
* Top customers by spending
* Best performing orders
* Daily sales trends
* Shipping time analysis
* Customer order frequency
